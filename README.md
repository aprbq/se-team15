# Install Frontend 
```sh
cd frontend
npm install
npm install lucide-react
npm run dev
```

# Github
```sh
git add . //เพิ่มไฟล์ทั้งหมดที่แก้หรือเพิ่มเตรียมคอมมิท
git commit -m "<ข้อความ>- close#<เลข task>" //ยืนยันการเปลี่ยนแปลงไฟล์พร้อมข้อความ
git push origin main //เพิ่มไฟล์ที่แก้หรือเพิ่มบน remote
git status //แสดงไฟล์ที่มีการแก้ไข จะบอก brach ด้วย
git branch //แสดง branch ทั้งหมด
git checkout //เปลี่ยน branch
git fetch //เช็คว่าไฟล์ local กับ remote ต่างกันไหม
git pull //ดึงไฟล์ใน remote ลงมาใน local
```
